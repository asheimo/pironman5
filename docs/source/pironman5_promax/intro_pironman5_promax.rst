Pironman 5 Pro MAX
================================================================================

Gracias por elegir nuestro |link_pironman5_promax|.

.. image:: img/Pironman-5-Pro-Max.png
    :width: 400
    :align: center

Eleve su experiencia con la Raspberry Pi 5 mediante el |link_pironman5_promax|, nuestro nuevo chasis de aluminio de alto rendimiento insignia. Diseñado para una refrigeración, expansión y una experiencia completa similar a la de un PC de sobremesa, es perfecto para aplicaciones exigentes como NAS, desarrollo de IA, centros multimedia y proyectos avanzados.

**Características Clave**:

* **Dual NVMe y Hub de Expansión para IA**: Conmutador PCIe Gen 2 integrado con dos ranuras M.2 M-key (2230/2242/2260/2280) para SSD o aceleradores de IA (totalmente compatible con Hailo-8/8L). Configure para NAS RAID 0/1, SSD+IA o configuraciones duales de IA.
* **Sistema de Refrigeración Integral**: Enfriador de torre masivo con ventilador PWM, más **tres** ventiladores RGB PWM direccionables, manteniendo un rendimiento térmico excelente incluso bajo carga máxima.
* **Doble Pantalla y Multimedia**:

    * **Pantalla Táctil DSI de 4.3"** (800x480): Úsela como pantalla de información secundaria, panel de estado o incluso como monitor principal de escritorio.
    * **Pantalla OLED Inteligente de 0.96"**: Muestra CPU, memoria, temperatura, uso de disco e IP en tiempo real.
    * **Altavoces Estéreo de 3W y Cámara de 5MP**: Listo para multimedia, videoconferencias o proyectos de visión por computadora.

* **Interfaz de Usuario y Control Mejorados**:

    * **Ecosistema RGB Personalizable**: Seis LEDs direccionables WS2812B y tres ventiladores RGB sincronizados para efectos de iluminación dinámicos.
    * **Receptor IR** para control remoto de centros multimedia (ej., Kodi, Volumio).
    * **Botón de Alimentación Metálico de Estilo Retro** para apagado e inicio seguros.
    * **Compartimento para Batería RTC** (para CR1220) para mantenimiento de tiempo.

* **Conectividad y Diseño Profesionales**:

    * **Reorganización del Panel Trasero**: La entrada de alimentación y los puertos HDMI estándar duales se reubican para alinearse con los puertos USB, ofreciendo una gestión de cables limpia y unificada.
    * **Acceso Completo a Puertos**: HDMI dual, LAN Gigabit, 2x USB 3.0, 2x USB 2.0.
    * **Extensor GPIO Externo Etiquetado** para facilitar la conexión de accesorios.

* **Diseño Industrial Premium**: Chasis robusto de aleación de aluminio oscuro con paneles laterales de acrílico oscuro, ranura para tarjeta microSD con mecanismo de resorte y una estética moderna inspirada en PC de sobremesa.

.. note::

  Se recomienda encarecidamente usar la fuente de alimentación oficial de Raspberry Pi de 27W o una alternativa de alta calidad compatible como la |link_sf_27w_supply| para el Pironman 5 Pro Max. Una alimentación insuficiente puede causar inestabilidad o reinicios de la Raspberry Pi 5, especialmente bajo alta carga con unidades NVMe y accesorios.

**Especificaciones**

* **Dimensiones**: 140.9 x 77.0 x 138.7 mm
* **Material**:

    * Chasis Principal: Aleación de Aluminio Anodizado Oscuro
    * Paneles Laterales: Acrílico Oscuro

* **Plataforma Soportada**: Raspberry Pi 5 (solo)
* **Entrada de Alimentación**: USB Type-C, 5V/5A (Mínimo 27W recomendado)
* **Interfaces y Puertos**:

    * **GPIO de 40 pines de Raspberry Pi** (extendido externamente con etiquetas claras)
    * **Ranura para Tarjeta MicroSD** (con mecanismo de expulsión por resorte)
    * **Panel Trasero**:

        * Entrada de Alimentación USB Type-C
        * 2 x USB 2.0
        * 2 x USB 3.0
        * Gigabit Ethernet (RJ45)
        * 2 x Puertos HDMI Estándar (Tipo A) (soporte 4Kp60)

* **Sistema de Refrigeración**:

    * 1 x Enfriador de Torre Grande con Ventilador Controlado por PWM
    * 3 x Ventiladores RGB PWM Direccionables (controlados por GPIO, sincronizables)

* **Pantalla y Multimedia**:

    * Pantalla Táctil Capacitiva DSI de 4.3 pulgadas (800 x 480 píxeles)
    * Pantalla OLED de 0.96 pulgadas (128x64) para estadísticas del sistema
    * Altavoces Estéreo de 3W
    * Soporte para Módulo de Cámara de 5 Megapíxeles

* **Almacenamiento y Expansión**:

    * Conmutador PCIe Gen 2 Integrado
    * 2 x Ranuras M.2 M-key PCIe 2.0 x1
    * Tamaños de SSD/Tarjeta AI Soportados: 2230, 2242, 2260, 2280

* **Controles, Iluminación y Características**:

    * 18 LEDs RGB direccionables WS2812B: 6 en la placa y 12 integrados en los ventiladores RGB.
    * Receptor IR (38kHz)
    * Botón de Alimentación Metálico (función de apagado seguro)
    * Soporte para Batería RTC (Reloj en Tiempo Real) (para celda CR1220)

* **Calidad de Construcción**: Cuerpo de aluminio CNC de precisión con ventanas de acrílico oscuro templado, diseñado para durabilidad y estética sofisticada.
