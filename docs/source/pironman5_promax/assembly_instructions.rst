2. Instructions d'assemblage
=============================================

Avant d'assembler le Pironman, veuillez d'abord vérifier que toutes les pièces et composants sont inclus. S'il manque des composants ou s'ils sont endommagés, veuillez contacter immédiatement SunFounder à l'adresse service@sunfounder.com pour résoudre le problème dès que possible.

**Conseils**

* Il est recommandé de suivre les instructions d'installation papier étape par étape.
* Avant l'installation, triez et empilez les composants pour les trouver plus rapidement.
* Vérifiez à plusieurs reprises les connexions des câbles FPC pour éviter un mauvais contact.
