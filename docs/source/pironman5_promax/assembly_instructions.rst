2. 組み立て手順
=============================================

Pironmanを組み立てる前に、まずすべての部品とコンポーネントが揃っていることを確認してください。不足している部品や破損しているコンポーネントがある場合は、すぐに service@sunfounder.com までSunFounderに連絡し、問題をできるだけ早く解決してください。

**ヒント**

* 紙の取扱説明書に従って、ステップバイステップで進めることをお勧めします。
* 組み立て前に、コンポーネントを分類して積み重ねておくと、より素早く見つけることができます。
* FPCケーブルの接続は、接触不良を避けるために繰り返し確認してください。
