2. 组装说明
=============================================

在开始组装 Pironman 之前，请先检查所有零件和组件是否齐全。  
如果发现有缺失或损坏的部件，请及时联系 SunFounder 官方支持：service@sunfounder.com 进行处理。

**提示**

* 建议按照纸质说明书逐步进行安装  
* 在安装前先将所有零件分类整理，方便快速查找  
* 多次检查 FPC 排线连接是否牢固，避免接触不良问题  

* :download:`[PDF] Pironman 5 Pro MAX 零件清单与组装说明 <Pironman5ProMAX.pdf>`
