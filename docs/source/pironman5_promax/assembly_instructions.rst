2. Instrucciones de Ensamblaje
=============================================

Antes de ensamblar el Pironman, verifique primero que todas las piezas y componentes estén incluidos. Si falta algún componente o está dañado, comuníquese con SunFounder inmediatamente en service@sunfounder.com para resolver el problema lo antes posible.

**Consejos**

* Se recomienda seguir las instrucciones de instalación en papel paso a paso.
* Antes de la instalación, clasifique y apile los componentes para encontrarlos más rápidamente.
* Verifique repetidamente las conexiones de los cables FPC para evitar mal contacto.

* :download:`[PDF]Lista de Componentes y Ensamblaje de Pironman 5 Pro MAX <Pironman5ProMAX.pdf>`
