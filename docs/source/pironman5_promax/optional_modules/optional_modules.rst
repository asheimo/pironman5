7. Modules optionnels
==============================================================

.. note::

    Les produits de la série Pironman 5 n'incluent pas les modules suivants.
    Vous devez les préparer vous-même ou les acheter sur notre site officiel :

    * `RTL-SDR Blog V4 <https://www.sunfounder.com/products/rtl-sdr-blog-v4>`_

Cette section explique comment étendre les capacités de la série Pironman 5 en intégrant des modules matériels optionnels, notamment un module caméra, un écran tactile 3,5 pouces et un récepteur radio RTL-SDR.
En combinant ces ajouts, vous pouvez améliorer l'interactivité, permettre une entrée visuelle en temps réel et débloquer une réception avancée de signaux sans fil.
