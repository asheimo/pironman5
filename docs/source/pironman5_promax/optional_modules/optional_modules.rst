7. オプションモジュール
==============================================================

.. note::

    Pironman 5シリーズ製品には、以下のモジュールは付属していません。
    これらはお客様自身でご用意いただくか、公式ウェブサイトからご購入ください：

    * `RTL-SDR Blog V4 <https://www.sunfounder.com/products/rtl-sdr-blog-v4>`_

このセクションでは、カメラモジュール、3.5インチタッチスクリーンディスプレイ、RTL-SDRラジオレシーバーなどのオプションのハードウェアモジュールを統合することで、Pironman 5シリーズの機能を拡張する方法について説明します。
これらのアドオンを組み合わせることで、インタラクティビティを高め、リアルタイムの視覚入力を可能にし、高度な無線信号受信の機能を引き出すことができます。
