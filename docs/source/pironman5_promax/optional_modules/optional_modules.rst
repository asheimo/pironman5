7. 可选模块
==============================================================

.. note::

   Pironman 5 系列产品 **不包含以下模块**。  
   需要用户自行准备，或从我们的官方网站购买：

   * `RTL-SDR Blog V4 <https://www.sunfounder.com/products/rtl-sdr-blog-v4>`_

本章节介绍如何通过集成可选硬件模块来扩展 Pironman 5 系列的功能，包括摄像头模块、3.5 英寸触摸屏以及 RTL-SDR 无线电接收器。  
通过这些扩展模块的组合，可以提升系统的交互能力，实现实时视觉输入，并支持高级无线信号接收功能。
