7. Optionale Module
==============================================================

.. note::

    Die Pironman-5-Serienprodukte enthalten die folgenden Module nicht.
    Sie müssen sie selbst vorbereiten oder auf unserer offiziellen Website erwerben:

    * `RTL-SDR Blog V4 <https://www.sunfounder.com/products/rtl-sdr-blog-v4>`_

In diesem Abschnitt wird erläutert, wie Sie die Fähigkeiten der Pironman-5-Serie durch die Integration optionaler Hardwaremodule erweitern können, darunter ein Kameramodul, ein 3,5-Zoll-Touchscreen-Display und einen RTL-SDR-Funkempfänger.
Durch die Kombination dieser Erweiterungen können Sie die Interaktivität verbessern, visuelle Eingaben in Echtzeit ermöglichen und den Empfang fortschrittlicher drahtloser Signale freischalten.
