Mini-USB-Mikrofon
===================================

.. image:: img/mini_usb_micphone.png
   :width: 300

**Beschreibung**

Dieses Plug-and-Play-Mini-USB-Mikrofon benötigt keinen Treiber, und dank seiner USB-Schnittstelle können Sie es mit jedem Computer, Laptop, Einplatinencomputer usw. verwenden!
Es funktioniert hervorragend mit einem Raspberry Pi, aber Sie können es auch mit jedem anderen Computer verwenden, mit dem Sie Audio aufnehmen möchten. Einfach einstecken und schon kann es losgehen mit der Aufnahme Ihres Podcasts, Ihres Liedes, Ihrer gesprochenen Poesie, Ihres Hörbuchs oder was auch immer Ihr Herz begehrt! Wenn Sie etwas näher an der Audioquelle sein müssen, können Sie es mit einem USB-Verlängerungskabel oder einem USB-Drehadapter kombinieren.
Bitte beachten Sie, dass dieses Mikrofon am besten in einem Anschluss verwendet wird, der sich nicht in der Nähe des Kühlgebläses Ihres Computers befindet.



**Merkmale**

* Funktioniert ohne Treiber: Einfach in Ihren Windows-PC einstecken und schon haben Sie ein sofort einsatzbereites Mikrofon
* Gut geeignet für Sprachdiktiersoftware oder Skype
* Tragbar, kann im Laptop/Notebook eingesteckt bleiben
* Das omnidirektionale Geräuschunterdrückungsmikrofon nimmt Töne aus größeren Entfernungen auf
* Abmessungen: 22,0 mm x 19,0 mm x 7,0 mm / 0,87" x 0,75" x 0,28"
* Gewicht: 3g / 0,11oz
* Schnittstellentyp: USB
