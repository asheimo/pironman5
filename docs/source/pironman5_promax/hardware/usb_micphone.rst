ミニ USB マイク
===================================

.. image:: img/mini_usb_micphone.png
   :width: 300

**説明**

このプラグアンドプレイのミニ USB マイクはドライバーを必要とせず、USB インターフェースにより、あらゆるコンピューター、ノートパソコン、SBC などで使用できます！
Raspberry Pi で非常に効果的に動作しますが、オーディオを録音したい他のコンピューターでも使用できます。単に接続するだけで、ポッドキャスト、歌、朗読詩、オーディオブックなど、お好みのものを録音する準備が整います！音源にもう少し近づく必要がある場合は、USB 延長ケーブルまたは USB スイベルアダプターと組み合わせるとよいでしょう。
このマイクは、コンピューターの冷却ファンの近くにないポートで使用するのが最適であることに注意してください。



**特徴**

* ドライバー不要：Windows PC に接続するだけで、すぐにマイクとして使用可能
* 音声入力ソフトウェアや Skype に最適
* ポータブル：ノートパソコンに接続したままにできる
* 全指向性ノイズキャンセリングマイクが遠距離からの音声も拾います
* 寸法： 22.0mm x 19.0mm x 7.0mm / 0.87インチ x 0.75インチ x 0.28インチ
* 重量： 3g / 0.11オンス
* インターフェースタイプ： USB
