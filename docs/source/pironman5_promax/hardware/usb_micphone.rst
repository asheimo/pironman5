迷你 USB 麦克风
===================================

.. image:: img/mini_usb_micphone.png
   :width: 300

**描述**

这是一款即插即用的迷你 USB 麦克风，无需驱动程序。其 USB 接口使其可用于任何电脑、笔记本、单板计算机（SBC）等设备。

它非常适合搭配 Raspberry Pi 使用，同时也兼容其他任意需要录音的计算机。只需插入设备即可开始录制播客、歌曲、有声读物、朗读内容或其他音频。如果需要更靠近声源，建议搭配 USB 延长线或 USB 旋转转接头使用。

请注意，建议将该麦克风插在远离电脑散热风扇的位置，以获得更好的录音效果。



**特性**

* 无需驱动：插入 Windows 电脑即可立即使用
* 适用于语音输入软件或 Skype 等应用
* 便携设计，可长期插在笔记本电脑上使用
* 全向降噪麦克风，可在较远距离拾取声音
* 尺寸：22.0mm × 19.0mm × 7.0mm / 0.87" × 0.75" × 0.28"
* 重量：3g / 0.11oz
* 接口类型：USB
