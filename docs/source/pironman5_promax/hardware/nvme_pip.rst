デュアルNVMe PIP
=====================


**デュアルNVMe PIP** （PCIe Peripheral Board）は、Raspberry Pi財団によって定義された、NVMeソリッドステートドライブ専用に設計されたPCIeアダプターです。

Raspberry Pi 5のPCIeインターフェースは、ネイティブで単一の **Gen2 x1** レーン（500 MB/s）を提供します。 **ASM1182e** PCIeスイッチチップを統合することにより、デュアルNVMe PIPはこれを **2つの独立したGen2 x1レーン** に拡張し、以下を接続できるようにします：

* **2台のM.2 NVMe SSD**、または
* **1台のM.2 NVMe SSD + 1台のM.2 Hailo-8/8L AIアクセラレーター**

**主な注意点**：

* Gen3はサポートされていません
* 対応NVMe SSDサイズ： **2230**、 **2242**、 **2260**、 **2280** （すべてM.2 M-keyスロット）

.. image:: img/nvme_pip.png

* ボードは、16P 0.5mmリバースFFC（フラットフレキシブルケーブル）またはカスタムインピーダンス整合FPC（フレキシブルプリント回路）ケーブルを介して接続します。
* **STA**： ステータスLEDインジケーター。
* **PWR**： 電源LEDインジケーター。
* オンボードの3.3V電源は最大3Aの出力をサポートできます。ただし、Raspberry Pi PCIeインターフェースは5V/1Aの出力（5Wに相当）を提供することに制限されているため、3.3V/3A使用のための追加電力は、J3コネクタを介して5V電源から供給できます。
* **FORCE ENABLE**： オンボード電源はPCIeインターフェースからのスイッチ信号によって起動します。Raspberry Piの電源が入った後、3.3V電源をオンにする信号を送信します。一部のシステムがスイッチ信号をサポートしていない場合やその他の理由により、2つの浮遊パッド間にワイヤーを半田付けしてJ4 FORCE ENABLEを短絡し、オンボード3.3V電源に強制的にNVMeに電力を供給させることができます。

モデルについて
---------------------------

コンパクトなサイズで知られるM.2 SSDは、主にキーイング（コネクタのノッチデザイン）と使用するインターフェースによって区別される様々なタイプがあります。主なタイプは以下の通りです：

* **M.2 SATA SSD**： これらはSATAインターフェースを使用し、2.5インチSATA SSDと同様ですが、より小型のM.2フォームファクタです。SATA IIIの最大速度である約600 MB/sに制限されます。これらのSSDは、BキーとMキー用にキーイングされたM.2スロットと互換性があります。
* **M.2 NVMe SSD**： これらのSSDはPCIeレーン上でNVMeプロトコルを使用し、M.2 SATA SSDよりも大幅に高速です。ゲーム、ビデオ編集、データ集約型タスクなど、高い読み取り/書き込み速度を必要とするアプリケーションに適しています。これらのSSDは通常、Mキーイングされたスロットを必要とします。これらのドライブはPCIe（Peripheral Component Interconnect Express）インターフェースを利用し、3.0、4.0、5.0などの異なるバージョンがあります。PCIeの新しいバージョンごとに、データ転送速度は実質的に前世代の2倍になります。ただし、Raspberry Pi 5はPCIe 3.0インターフェースを使用しており、最大3,500 MB/sの転送速度を提供できます。

M.2 SSDには、Bキー、Mキー、B+Mキーの3つのキータイプがあります。しかし、その後、BキーとMキーの機能を組み合わせたB+Mキーが登場しました。その結果、単独のBキーに取って代わりました。下の画像を参照してください。

.. image:: img/ssd_key.png


一般的に、M.2 SATA SSDはB+Mキーイング（BキーイングおよびMキーイングモジュール用のソケットに適合可能）であり、PCIe 3.0 x4レーン用のM.2 NVMe SSDはMキーイングです。

.. image:: img/ssd_model2.png

長さについて
-----------------------

M.2モジュールには様々なサイズがあり、Wi-Fi、WWAN、Bluetooth、GPS、NFCにも利用できます。

Pironman 5 MAXは、その名称に基づいて4つの（PCIe Gen 2.0）NVMe M.2 SSDサイズをサポートしています： 2230、2242、2260、2280。「22」は幅（ミリメートル）、その後の2桁は長さです。ドライブが長いほど、より多くのNANDフラッシュチップを搭載できるため、より大容量になります。


.. image:: img/m2_ssd_size.png
  :width: 600


電源
-----------------------

オンボードのデュアル3.3V電源は、最大3A（10W）の出力をサポートします。両方の電源レールは干渉することなく独立して動作します。

**FORCE ENABLE**
オンボード電源はPCIeインターフェースからのスイッチ信号によって起動します。Raspberry Piが起動すると、信号が3.3V電源をオンにします。システムがスイッチ信号をサポートしていない場合やその他の理由により、J4 FORCE ENジャンパーを短絡して、オンボード3.3V電源を強制的にNVMe用に有効にします。

**LED**
各インターフェースには、独立した電源ステータスインジケーター（PWR）と状態インジケーター（STA）があります。

電源スイッチコンバーター
-------------------------------


**電源ボタンの追加**

* Raspberry Pi 5には、RTCバッテリーコネクタとボードエッジの間に位置する **J2** ジャンパーがあります。このブレイクアウトにより、2つのパッド間にノーマリーオープン（NO）モーメンタリースイッチを接続することで、Raspberry Pi 5にカスタム電源ボタンを追加できます。このスイッチを短く押すと、オンボード電源ボタンの機能を模倣します。

   .. image:: img/pi5_j2.jpg

* Pironman 5には、 **J2** ジャンパーを2つのポゴピンを使用して外部電源ボタンに拡張する **電源スイッチコンバーター** があります。

   .. image:: img/psc.png

* これで、Raspberry Pi 5の電源オン/オフを電源ボタンを使用して行えるようになりました。

**電源サイクル**

Raspberry Pi 5に最初に電源を投入すると、ボタンを押す必要なく自動的に電源がオンになり、オペレーティングシステムが起動します。

Raspberry Pi Desktopを実行している場合、電源ボタンを短く押すと、クリーンシャットダウンプロセスが開始されます。シャットダウン、再起動、またはログアウトのオプションを提供するメニューが表示されます。オプションを選択するか、電源ボタンをもう一度押すと、クリーンシャットダウンが開始されます。

.. image:: img/button_shutdown.png

**シャットダウン**

    * **Raspberry Pi OS Desktop** システムを実行している場合は、電源ボタンを素早く2回押すとシャットダウンします。
    * デスクトップのない **Raspberry Pi OS Lite** システムを実行している場合は、電源ボタンを1回押すとシャットダウンが開始されます。
    * 強制的なハードシャットダウンを行うには、電源ボタンを長押しします。


**電源オン**

    * Raspberry Piボードがシャットダウンしているが、電源が供給されている状態の場合、シャットダウン状態から電源を入れるには1回押します。

.. note::

    シャットダウンボタンをサポートしていないシステムを実行している場合は、5秒間長押しすると強制的にハードシャットダウンし、シャットダウン状態から1回押すと電源が入ります。
