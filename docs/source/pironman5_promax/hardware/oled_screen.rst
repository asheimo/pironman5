Pantalla OLED
===================

.. image:: img/oled_screen.png
    :width: 400

* **Tamaño**: 0.96''
* **Material**: PM OLED
* **Color**: Luz blanca
* **Controlador**: SSD1306
* **Voltaje**: 3.3V
* **Resolución**: 128x64
* **Área de visualización**: 21.74 x 10.86 mm
* **Tamaño del panel**: 26.70 x 19.26 x 1.42 mm
* **Paso de píxel**: 0.17 x 0.17 mm
* **Tamaño de píxel**: 0.154 x 0.154 mm
* **Ángulo de visión**: Visión completa
* **Temperatura de funcionamiento**: -20 ~ 70°C
* **Método de comunicación**: IIC/SPI/Parallel
* **Método de conexión**: FPC tipo enchufe de paso 0.5 mm
