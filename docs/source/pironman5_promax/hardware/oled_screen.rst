Écran OLED
===================

.. image:: img/oled_screen.png
    :width: 400

* **Taille** : 0,96 pouce
* **Matériau** : PM OLED
* **Couleur** : Lumière blanche
* **Pilote** : SSD1306
* **Tension** : 3,3V
* **Résolution** : 128x64
* **Zone d'affichage** : 21,74 x 10,86 mm
* **Taille du panneau** : 26,70 x 19,26 x 1,42 mm
* **Pas de pixel** : 0,17 x 0,17 mm
* **Taille de pixel** : 0,154 x 0,154 mm
* **Angle de vue** : Vue complète
* **Température de fonctionnement** : -20 à 70°C
* **Méthode de communication** : IIC/SPI/Parallèle
* **Méthode de connexion** : FPC enfichable pas à pas 0,5 mm
