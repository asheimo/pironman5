Características
======================

**Especificaciones**

* **Dimensiones**: 140.9 x 77.0 x 138.7 mm
* **Material**:

    * Chasis Principal: Aleación de Aluminio Anodizado Oscuro
    * Paneles Laterales: Acrílico Oscuro

* **Plataforma Soportada**: Raspberry Pi 5 (solo)
* **Entrada de Alimentación**: USB Type-C, 5V/5A (Mínimo 27W recomendado)
* **Interfaces y Puertos**:

    * **GPIO de 40 pines de Raspberry Pi** (extendido externamente con etiquetas claras)
    * **Ranura para Tarjeta MicroSD** (con mecanismo de expulsión por resorte)
    * **Panel Trasero**:

        * Entrada de Alimentación USB Type-C
        * 2 x USB 2.0
        * 2 x USB 3.0
        * Gigabit Ethernet (RJ45)
        * 2 x Puertos HDMI Estándar (Tipo A) (soporte 4Kp60)

* **Sistema de Refrigeración**:

    * 1 x Enfriador de Torre Grande con Ventilador Controlado por PWM
    * 3 x Ventiladores PWM RGB Direccionables (controlados por GPIO, sincronizables)

* **Pantalla y Multimedia**:

    * Pantalla Táctil Capacitiva DSI de 4.3 pulgadas (800 x 480 píxeles)
    * Pantalla OLED de 0.96 pulgadas (128x64) para estadísticas del sistema
    * Altavoces Estéreo de 3W
    * Soporte para Módulo de Cámara de 5 Megapíxeles

* **Almacenamiento y Expansión**:

    * Conmutador PCIe Gen 2 Integrado
    * 2 x Ranuras M.2 M-key PCIe 2.0 x1
    * Tamaños de SSD/Tarjeta AI Soportados: 2230, 2242, 2260, 2280

* **Controles, Iluminación y Características**:

    * 18 LEDs RGB direccionables WS2812B: 6 en la placa y 12 integrados en los ventiladores RGB.
    * Receptor IR (38kHz)
    * Botón de Alimentación Metálico (función de apagado seguro)
    * Soporte para Batería RTC (Reloj en Tiempo Real) (para celda CR1220)

* **Calidad de Construcción**: Cuerpo de aluminio CNC de precisión con ventanas de acrílico oscuro templado, diseñado para durabilidad y estética sofisticada.
