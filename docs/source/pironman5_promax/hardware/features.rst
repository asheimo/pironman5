Caractéristiques
======================

**Spécifications**

* **Dimensions** : 140,9 x 77,0 x 138,7 mm
* **Matériau** :

    * Châssis principal : Alliage d'aluminium anodisé foncé
    * Panneaux latéraux : Acrylique foncé

* **Plateforme supportée** : Raspberry Pi 5 (uniquement)
* **Alimentation électrique** : USB Type-C, 5V/5A (27W minimum recommandé)
* **Interfaces et ports** :

    * **GPIO 40 broches du Raspberry Pi** (étendu extérieurement avec étiquettes claires)
    * **Emplacement pour carte MicroSD** (avec mécanisme d'éjection à ressort)
    * **Panneau arrière** :

        * Entrée d'alimentation USB Type-C
        * 2 x USB 2.0
        * 2 x USB 3.0
        * Ethernet Gigabit (RJ45)
        * 2 x Ports HDMI standard (Type A) (support 4Kp60)

* **Système de refroidissement** :

    * 1 x Grand refroidisseur tour avec ventilateur contrôlé par PWM
    * 3 x Ventilateurs PWM RGB adressables (contrôlés par GPIO, synchronisables)

* **Affichage et multimédia** :

    * Écran tactile capacitif DSI 4,3 pouces (800 x 480 pixels)
    * Écran OLED 0,96 pouce (128x64) pour les statistiques système
    * Haut-parleurs stéréo 3W
    * Support du module caméra 5 mégapixels

* **Stockage et expansion** :

    * Commutateur PCIe Gen 2 intégré
    * 2 x Fentes M.2 M-key PCIe 2.0 x1
    * Tailles SSD/carte AI supportées : 2230, 2242, 2260, 2280

* **Commandes, éclairage et caractéristiques** :

    * 18 LEDs RGB adressables WS2812B : 6 sur la carte et 12 intégrées dans les ventilateurs RGB.
    * Récepteur IR (38 kHz)
    * Bouton d'alimentation métallique (fonction d'arrêt sécurisé)
    * Support de batterie pour horloge temps réel (RTC) (pour pile CR1220)

* **Qualité de fabrication** : Corps en aluminium CNC de précision avec fenêtres en acrylique foncé trempé, conçu pour la durabilité et une esthétique sophistiquée.
