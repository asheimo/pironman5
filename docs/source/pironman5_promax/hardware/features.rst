Features
======================


**Specifications**

* **Dimensions**: 140.9 x 77.0 x 138.7 mm 
* **Material**:

    * Main Chassis: Dark Anodized Aluminum Alloy
    * Side Panels: Dark Acrylic

* **Supported Platform**: Raspberry Pi 5 (only)
* **Power Input**: USB Type-C, 5V/5A (Minimum 27W recommended)
* **Interfaces & Ports**:

    * **Raspberry Pi 40-pin GPIO** (externally extended with clear labels)
    * **MicroSD Card Slot** (with spring-loaded eject mechanism)
    * **Rear Panel**:

        * USB Type-C Power Input
        * 2 x USB 2.0
        * 2 x USB 3.0
        * Gigabit Ethernet (RJ45)
        * 2 x Standard HDMI (Type A) Ports (4Kp60 support)

* **Cooling System**:

    * 1 x Large Tower Cooler with PWM-controlled Fan
    * 3 x  Addressable RGB PWM Fans (GPIO controlled, synchronizable)

* **Display & Multimedia**:

    * 4.3-inch DSI Capacitive Touchscreen (800 x 480 pixels)
    * 0.96-inch OLED Display (128x64) for system stats
    * Stereo 3W Speakers
    * 5 Megapixel Camera Module support

* **Storage & Expansion**:

    * Integrated PCIe Gen 2 Switch
    * 2 x PCIe 2.0 x1 M.2 M-key Slots
    * Supported SSD/AI Card Sizes: 2230, 2242, 2260, 2280

* **Controls, Lighting & Features**:

    * 6 x WS2812B Addressable RGB LEDs
    * IR Receiver (38kHz)
    * Metal Power Button (safe shutdown function)
    * Real-Time Clock (RTC) Battery Holder (for CR1220 cell)

* **Build Quality**: Precision CNC aluminum body with tempered dark acrylic windows, designed for durability and sophisticated aesthetics.
