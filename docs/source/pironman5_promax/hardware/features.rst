特徴
======================


**仕様**

* **寸法**: 140.9 x 77.0 x 138.7 mm
* **材質**:

    * メインフレーム： ダークアルマイト処理アルミニウム合金
    * サイドパネル： ダークアクリル

* **対応プラットフォーム**: Raspberry Pi 5（のみ）
* **電源入力**: USB Type-C、5V/5A（最低27W推奨）
* **インターフェースとポート**:

    * **Raspberry Pi 40ピン GPIO** （外部に拡張、明確なラベル付き）
    * **MicroSDカードスロット** （スプリング式イジェクト機構付き）
    * **背面パネル**:

        * USB Type-C 電源入力
        * 2 x USB 2.0
        * 2 x USB 3.0
        * ギガビットイーサネット（RJ45）
        * 2 x 標準 HDMI（Type A）ポート（4Kp60 対応）

* **冷却システム**:

    * 1 x 大型タワークーラー（PWM制御ファン付き）
    * 3 x アドレッサブル RGB PWM ファン（GPIO制御、同期可能）

* **ディスプレイとマルチメディア**:

    * 4.3インチ DSI 静電容量式タッチスクリーン（800 x 480 ピクセル）
    * 0.96インチ OLED ディスプレイ（128x64）、システム情報表示用
    * ステレオ 3W スピーカー
    * 5 メガピクセル カメラモジュール 対応

* **ストレージと拡張**:

    * 統合 PCIe Gen 2 スイッチ
    * 2 x PCIe 2.0 x1 M.2 M-key スロット
    * 対応 SSD/AI カードサイズ： 2230、2242、2260、2280

* **コントロール、照明と機能**:

    * このボードには、18個のWS2812BアドレッサブルRGB LEDが搭載されています。内訳は、6個がオンボード、12個がRGBファンに統合されており、カスタマイズ可能な制御を実現します。
    * IR レシーバー（38kHz）
    * メタル製電源ボタン（安全シャットダウン機能）
    * リアルタイムクロック（RTC）バッテリーホルダー（CR1220 セル用）

* **ビルド品質**: 精密 CNC 加工アルミボディにテンパードダークアクリルウィンドウを採用、耐久性と洗練された美観を実現。
