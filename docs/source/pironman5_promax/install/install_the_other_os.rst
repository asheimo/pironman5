Ubuntu/Kali Linux/Homebridgeのインストール
=========================================================================

お手持ちのMicro SDカードまたはNVMe SSDに応じて、インストール方法を選択できます。

**Micro SDカードのみを使用する場合**

  Micro SDカードを使用する場合は、以下の最初の方法に従ってください。

**M.2 NVMe SSDを使用する場合**

  * M.2 NVMe SSDを使用する場合は、SSDをコンピューターに接続するための **M.2 NVMe SSDエンクロージャアダプター** も必要です。2番目の方法に従ってOSをインストールしてください。
