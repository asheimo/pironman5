Ubuntu/Kali Linux/Homebridge installieren
=========================================================================

Sie können die Installationsmethode basierend auf der Verfügbarkeit einer Micro-SD-Karte oder einer NVMe-SSD wählen.

**Nur mit einer Micro-SD-Karte**

  Wenn Sie eine Micro-SD-Karte verwenden, können Sie einfach der ersten Methode unten folgen.

**Mit einer M.2-NVMe-SSD**

  * Wenn Sie eine M.2-NVMe-SSD verwenden, benötigen Sie zusätzlich einen **M.2-NVMe-SSD-Gehäuseadapter**, um Ihre SSD mit Ihrem Computer zu verbinden, und folgen dann der zweiten Methode, um das Betriebssystem zu installieren.
