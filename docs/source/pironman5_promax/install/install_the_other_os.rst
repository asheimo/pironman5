安装 Ubuntu / Kali Linux / Homebridge
=========================================================================

你可以根据是否拥有 Micro SD 卡或 NVMe SSD，选择不同的安装方式。

**仅使用 Micro SD 卡**

  如果使用 Micro SD 卡，可以直接按照下面的第一种方法进行安装。

**使用 M.2 NVMe SSD**

  * 如果使用 M.2 NVMe SSD，则需要一个 **M.2 NVMe SSD 外接盒（Enclosure Adapter）** 将 SSD 连接到电脑，然后按照第二种方法安装操作系统。
