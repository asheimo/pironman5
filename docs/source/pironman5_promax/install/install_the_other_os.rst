Installazione di Ubuntu/Kali Linux/Homebridge
=========================================================================

Puoi scegliere il metodo di installazione in base al fatto che tu abbia a disposizione una Micro SD o un SSD NVMe.

**Utilizzare Solo una Scheda Micro SD**

  Se stai utilizzando una scheda Micro SD, puoi semplicemente seguire il primo metodo qui sotto.

**Utilizzare un SSD M.2 NVMe**

  * Se stai utilizzando un SSD M.2 NVMe, avrai bisogno anche di un **Adattatore per Enclosure M.2 NVMe** per collegare il tuo SSD al computer e seguire il secondo metodo per installare il sistema operativo.
