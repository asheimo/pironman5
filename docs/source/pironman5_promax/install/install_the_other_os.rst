Installation d'Ubuntu/Kali Linux/Homebridge
=========================================================================

Vous pouvez choisir la méthode d'installation selon que vous disposez d'une carte Micro SD ou d'un SSD NVMe.

**Utilisation uniquement d'une carte Micro SD**

  Si vous utilisez une carte Micro SD, vous pouvez simplement suivre la première méthode ci-dessous.

**Utilisation d'un SSD M.2 NVMe**

  * Si vous utilisez un SSD M.2 NVMe, vous aurez également besoin d'un **adaptateur pour boîtier SSD M.2 NVMe** pour connecter votre SSD à votre ordinateur et suivre la deuxième méthode pour installer le système d'exploitation.
