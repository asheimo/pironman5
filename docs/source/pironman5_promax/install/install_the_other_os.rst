Instalación de Ubuntu/Kali Linux/Homebridge
=========================================================================

Puede elegir el método de instalación según si tiene a mano una Micro SD o un SSD NVMe.

**Usar Solo una Tarjeta Micro SD**

  Si está usando una tarjeta Micro SD, simplemente puede seguir el primer método a continuación.

**Usar un SSD M.2 NVMe**

  * Si está usando un SSD M.2 NVMe, también necesitará un **Adaptador de Carcasa para SSD M.2 NVMe** para conectar su SSD a su ordenador y seguir el segundo método para instalar el SO.
