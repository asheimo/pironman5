5. Control mediante Comandos o Panel
=======================================================

Una vez que haya instalado correctamente el módulo ``pironman5``, el servicio ``pironman5.service`` se iniciará automáticamente al reiniciar.

Puede monitorear y controlar el Pironman 5 Pro MAX mediante comandos, o accediendo al panel a través de la página web en ``http://<ip>:34001``.
