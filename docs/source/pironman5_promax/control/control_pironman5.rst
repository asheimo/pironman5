5. Contrôle via Commandes ou Tableau de Bord
=======================================================

Une fois que vous avez installé avec succès le module ``pironman5``, le ``pironman5.service`` démarrera automatiquement au redémarrage.

Vous pouvez surveiller et contrôler le Pironman 5 Pro MAX via des commandes, ou en accédant au tableau de bord via la page web à l'adresse ``http://<ip>:34001``.
