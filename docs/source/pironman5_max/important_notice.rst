1. Retirez le cavalier de l'interrupteur à vibration
--------------------------------------------------------

À partir de la version v1.3.6, le réveil de l'écran OLED utilise le bouton d'alimentation. Vous devez retirer le cavalier de l'interrupteur à vibration pour éviter d'occuper les broches GPIO du Raspberry Pi et prévenir d'éventuels conflits.

.. note::
   Veuillez vérifier si ce cavalier est présent ; si ce n'est pas le cas, veuillez ignorer cet avis.
