1. Remove vibration Switch Jumper
------------------------------------------------

From v1.3.6, OLED wake-up uses the power button. You must remove the vibration switch jumper to avoid occupying Raspberry Pi GPIO pins and prevent potential conflicts. 

.. note:: 
   Please check whether this jumper exists; if not, please ignore this notice.
