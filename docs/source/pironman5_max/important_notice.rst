1. 振動スイッチジャンパーの取り外し
------------------------------------------------

v1.3.6 以降、OLEDウェイクアップは電源ボタンを使用します。Raspberry PiのGPIOピンを占有して競合が発生する可能性を防ぐため、振動スイッチジャンパーを必ず取り外してください。

.. note:: 
   このジャンパーが実装されているかご確認ください。存在しない場合は、本通知を無視してください。
