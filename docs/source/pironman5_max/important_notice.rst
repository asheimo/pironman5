1. Rimuovere il ponticello dell'interruttore a vibrazione
----------------------------------------------------------------------------------

Dalla versione v1.3.6, il risveglio del display OLED utilizza il pulsante di accensione. È necessario rimuovere il ponticello dell'interruttore a vibrazione per evitare di occupare i pin GPIO del Raspberry Pi e prevenire potenziali conflitti.

.. note::
   Verificare se questo ponticello è presente; in caso contrario, ignorare questo avviso.
