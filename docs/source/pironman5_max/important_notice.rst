1. 移除振动开关跳线
-----------------------------------------------

从 v1.3.6 版本开始，OLED 唤醒功能使用电源按键。您必须移除振动开关跳线，以避免占用 Raspberry Pi GPIO 引脚并防止潜在的冲突。

.. 注意：:
   请检查是否存在此跳线；如果没有，请忽略本通知。
